# Money Game

본 프로젝트는 주식 거래, 은행 예적금 및 대출 관리, 카지노 미니게임을 포함하는 웹 기반 자산 관리 시뮬레이션 게임입니다.

## 주요 기능 및 시스템 사양

### 1. 주식 시장 시스템
- **거래 시간 제한:** 게임 내 시간 기준 09:00부터 19:59까지만 거래가 가능하도록 제한되어 있습니다.
- **타임라인 인디케이터:** 현재 시간과 거래 가능한 윈도우를 시각적으로 나타내는 진행바가 제공됩니다.
- **단축 거래 지원:** 매수 및 매도 입력창이 비어있을 때 더블 클릭을 수행하면 현재 보유 자산(또는 주식 수) 기반 최대 수량이 자동으로 입력됩니다.
- **차트 상호작용:** 주식 상세 차트 영역을 더블 클릭할 경우, 해당 주식의 보유 수량 전체가 즉시 매도 처리됩니다.
- **일중 차트 (Intraday Chart):** 주식 목록에서 09:00부터 20:00 사이의 주가 변동을 보여주는 스파크라인 형태의 실시간 차트를 지원합니다.
- **가격 변동 알고리즘:** 인위적인 평균 회귀(Mean-reversion) 없이 순수 무작위 보행(Random Walk) 모델을 기반으로 주가 변동이 산출됩니다.

### 2. 은행 및 자산 관리
- **입력값 자동 보정:** 보유 현금 또는 상환 가능 금액을 초과하는 값을 입력할 경우, 허용 가능한 최대 수치로 입력 폼이 자동 보정됩니다.
- **통합 자산 추적:** 대시보드 뷰에서 (현금 + 예금 - 대출금 + 주식 평가액) 공식을 통해 총 자산 가치를 실시간으로 계산하여 표시합니다.

### 3. 레이아웃 및 그래픽 테마
- **고정 헤더 (Sticky Header):** 모든 페이지 상단에 플레이어의 현재 현금 잔고와 게임 내 동기화된 시각이 고정 노출됩니다.
- **UI/UX 테마:** 입체감이 부여된 3D 셰이프 버튼 및 굵은 테두리(Thick borders)를 사용하여, 시뮬레이션 게임에 적합한 단색 형태의 캐주얼 테마를 적용했습니다.

## 코드베이스 구조

유지보수 및 코드 확장성을 확보하기 위해 컴포넌트와 비즈니스 로직을 도메인 단위로 모듈화했습니다.

```text
src/
 ┣ components/
 ┃ ┣ ui/                  # 재사용 가능한 공통 UI 모듈 (BetButton, Card 등)
 ┃ ┣ casino/              # 카지노 게임 도메인 모듈 (BlackjackGame, SlotMachineGame)
 ┃ ┗ stock/               # 주식 시장 도메인 모듈 (차트 및 리스트)
 ┣ pages/                 # 컴포넌트 결합 및 라우팅을 위한 페이지 컨테이너 (Casino, StockMarket 등)
 ┗ utils/                 # 순수 비즈니스 로직 및 유틸리티 함수 (cards.js 등)
```
